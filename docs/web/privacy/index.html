Life App 개인정보 처리방침

최종 업데이트: 2025-10-12

Life App(이하 “회사”)는 오프라인 중심 웰니스 도구로서 Google Play 내부 테스트(Stage 0)를 통해 배포되고 있습니다. 본 방침은 테스트 기간 동안 사용자의 데이터를 어떻게 처리하는지 설명하며, 정식 출시 시 업데이트 후 공지됩니다.

1. 수집·처리하는 데이터

데이터 저장 위치 비고
타이머, 저널, 설정 정보 사용자 기기 Stage 0에서는 서버 동기화 없음
암호화 백업 파일 사용자가 선택한 저장소(예: Google Drive) 사용자가 백업을 수행할 때만 생성되며 회사는 복사본을 보관하지 않음
수면 사운드 녹음 로컬(최대 3일 보관) 휴식 요약을 생성한 후 자동 삭제 가능
크래시/분석 로그 비활성화 Stage 0에서는 원격 분석을 수집하지 않음

2. 데이터 이용 목적

  • 오프라인 타이머, 라이프 버디 코멘트, 수면/운동 루틴 기능 제공
  • 사용자가 요청한 암호화 백업·복원 지원
  • 선택 기능(수면 사운드 분석 등)에 필요한 임시 데이터 처리

3. 선택적 동기화 및 제3자 서비스

  • Google Drive – 사용자가 백업을 생성할 때에만 해당 파일을 업로드하며 회사는 해당 파일 내용에 접근하지 않습니다.
  • Google Play Console – 내부 테스트 배포 및 설치 통계를 위해 Google이 제공하는 정보를 사용할 수 있습니다.
  • Stage 0에서는 광고 네트워크 혹은 분석 도구와 데이터를 공유하지 않습니다.

4. 사용자의 선택권

  • 앱 내 설정 ▸ 백업/복원에서 백업 생성·삭제·복원을 직접 관리할 수 있습니다.
  • 디바이스 설정에서 앱 데이터를 삭제하거나 앱을 제거하면 모든 로컬 데이터가 삭제됩니다.
  • 수면 사운드 저장, Drive 백업 등은 모두 명시적 동의를 거쳐 사용하도록 설계되어 있습니다.

5. 보안

  • 백업 파일은 AES-256으로 암호화되며, 사용자만이 키/암호를 관리합니다.
  • 안전한 저장소(본인 소유의 암호화 스토리지 등)에 백업을 보관할 것을 권장합니다.

6. 아동 보호

Life App은 만 13세 미만 아동을 대상으로 하지 않으며, 해당 연령 미만 아동의 데이터를 의도적으로 수집하지 않습니다.

7. 정책 변경

서비스 단계가 변경되면 본 방침을 업데이트하고 앱 내 공지 및 사이트에 최신 정보를 게시합니다. “최종 업데이트” 일자가 반영됩니다.

8. 문의처

데이터 관련 문의 및 요청은 support@lifeapp.dev 로 연락해 주세요.
